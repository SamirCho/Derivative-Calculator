Derivative Calculator

Calculating derivative with respect to x; other variables are considered as constants.

Input field should be an algebriac expression in standard form, such as, Ax^2 + Bx + C. Negative exponents should be written as Ax^-B, where A is a coefficient and B is part of the exponent of the variable.

Select a function that will be applied to the entered expression, or press "no function" to evaluate just the expression itself.

Trigonometric functions are assumed to be radians.

The last two buttons allow you to customize the base of logarithmic and exponential functions. They are set as 2 by default. The natural logarithm and exponential functions also available.
