Derivative Calculator

Calculating derivative with respect to x; other variables are considered as constants.

Input field should be an algebriac expression in standard form, such as, Ax^2 + Bx + C. Negative exponents are also supported and should be written as Ax^-B, where A is a coefficient and B is part of the exponent of the variable. Rational expressions should be written as negative exponents and fractions should be converted to decimals.

Select a function that will be applied to the entered expression, or press "no function" to evaluate just the expression itself.

Trigonometric functions are assumed to be radians.

Customize the base of logarithmic and exponential functions here: They are set as 10 by default. The natural logarithm and exponential functions are also available.

Due to the way binary floating-point numbers store decimal values, decimals must be rounded. To be as accurate as possible indicate how many decimals you intend on using here: You do not need to worry about this (leave as 0) if you are using only integers.
